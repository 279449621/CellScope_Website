Installation
========================

If you use **CellScope** in another project, you can directly download it from `Github <https://github.com/zhigang-yao/CellScope>`_. Additionally, you can pull CellScope from PyPI with the following command:

**Python Version Requirement**

Ensure that you have Python version **3.7** or higher but lower than **3.11** installed.

**Pip/PyPI**

If your Python version is between **3.7** and **3.11**, you can install CellScope from PyPI by running the following command:

.. code-block:: bash

    $ pip install CellScope_RNA
